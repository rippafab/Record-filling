# Recovering Missing Data with Record Analysis

This repository contains the numerical codes used in the article *Recovering Missing Data with Record Analysis*.

The folder "constant_rate" contains the script and the source codes used for the synthetic test with constant event rate and time-dependent detection threshold. These codes are used to generate the first figures of the article.

The folder "omori" contains the script and the source codes used for the synthetic case study. The codes generate the complete and incomplete catalogs, apply the record-based refilling procedure, and produce the files used to compare the complete, observed, and reconstructed catalogs.
